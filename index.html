About us

Have you always wanted to build your first own website, but didn't know where to start? Or maybe you thought you weren't technical enough? Then these two days are for you! We'll take the first few steps down the web development road together. And here comes the best part: Itâ€™s completely free!

NEW DEVS ON THE BLOCK is a two-day workshop for web development made and supported by the beautiful people of the web community.

We'll teach you the fundamentals and guide you through coding your first website.

Schedule

The workshop is divided into different topics that build on top of each other. Each topic contains a theoretical part, where we go over some slides, and a coding part, where you will turn theory into code.

Topics

By the end of the workshop, you'll learn about or build the following:

  • History of the Web
  • Basic structure of a website
  • Styling (i.e. fonts and colors) your website
  • Working with images
  • Creating layouts
  • Minimal animations
  • Rotating image gallery

Technical Topics

We'll cover the following technical aspects throughout the workshop:

HTML

  • Semantic HTML & Accessibility
  • Headings and document structure
  • Sectional elements
  • Text elements

CSS

  • Selectors
  • Pseudo-classes and elements
  • Colors
  • Flexbox
  • Transitions

JavaScript

  • Flow Control
  • Types
  • Functions
  • DOM API
