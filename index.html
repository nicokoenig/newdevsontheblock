About us

Have you always wanted to build your first own website, but somehow, you didn't know how to get started? Maybe because you thought you were not that techie after all? Then these two days are precisely for you! We take the first few steps down the web development road together.

NEW DEVS ON THE BLOCK is a two-day workshop for front-end development made and supported by the beautiful people of the web community.

We will help you learn the fundamental building blocks of websites and guide you through coding your first website yourself.

Schedule

The workshop is divided into different topics that build on top of each other. Each topic contains a theoretical part, where we go over some slides, and a coding part, where you will turn theory into code.

Topics

You will learn the following things

  • History of the Web
  • Create the basic structure of a website
  • Add styling (i.e. fonts and colors) to your website
  • How to add images to your website
  • How to create layouts
  • How to add minimal animations
  • Use JavaScript to create a image gallery

Technical Topics

We will cover the following technical aspects throughout the workshop.

HTML

  • Semantic HTML & Accessibility
  • Headings and document structure
  • Sectional elements
  • Text elements

CSS

  • Selectors
  • Pseudo-classes and elements
  • Colors
  • Flexbox
  • Transitions

JavaScript

  • Types
  • Functions
  • DOM API
